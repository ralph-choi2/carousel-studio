# IG Caption Writer

Uphone(@uphone.english) 인스타그램 캡션 생성. 200개 포스트 성과 분석 기반 골든 템플릿.

## Input

- **필수**: 콘텐츠 설명 (JSON data 파일 또는 소재명), 포맷 (릴스/카드뉴스/피드)
- **선택**: 테마, 영어 표현 목록, 타겟 상황

JSON data 파일 경로가 주어지면 pages 내용에서 표현/상황을 자동 추출.

## 포맷 선택 가이드

| 콘텐츠 | 템플릿 |
|--------|--------|
| 릴스/숏폼 영상 | [A] 핵심표현 3가지 |
| 표현 하나 딥다이브 캐러셀 | [B] 단일 표현 카드 |
| 표현 3~5개 묶음 캐러셀 | [C] .zip 모음집 |
| 피드 이미지 1장 | [B] 단일 표현 카드 |

## 훅 라인 공식 (고성과 상위 20%)

| 패턴 | 예시 |
|------|------|
| 질문형 + 공감 | "~하고 있지 않나요?" |
| "X 말고 Y" | `"I forgot" 말고 더 부드럽게 말해보세요` |
| 놀라움/발견 | `이게 무례한 표현이라고?` |

**고성과 키워드**: "~다고요?", "말고", "이렇게", "해보세요 👇", "알고 계셨나요?"

## 규칙

- **해시태그**: 총 7개 (고정 5 + 주제별 2)
  - 고정: `#민병철유폰 #전화영어 #화상영어 #영어회화 #영어표현`
- **이모지**: 캡션 전체 최대 3개 (훅 1 + CTA 1 + 본문 0~1)
- **길이**: 300~400자 (최소 200, 최대 500)
- **톤**: 친근한 존댓말 (~해요, ~해보세요)
- **금지**: 8개 이상 해시태그, "시작해보세요/지금 바로" 광고 톤, "쉽다/빠르다" 약속

### 브랜드 보이스

- 2026: "영어회화, 이번엔 끝까지 간다" — 완주 구조 강조
- AI = 실전 준비 도구, 완성 = 외국인강사 1:1 실전
- 유폰 = 실전 파트너 (동기부여자 아님)

## Output

캡션 텍스트를 바로 복사 가능한 형태로 출력.

생성 후 JSON data 파일의 `caption` 필드에 저장:

```bash
# data 파일 직접 업데이트
# caption 필드에 생성된 텍스트 입력
```

시트 등록은 upload 스킬이 담당.
